## 🧠 Deep Learning & Computer Vision
*Located in: [`Deep_Learning_Projects/`](./Deep_Learning_Projects)*

This section contains projects focused on neural networks, computer vision, and deep learning architectures (CNNs, MLPs, etc.).

*   **`Animal_Image_classification.ipynb`**: Image classification pipeline designed to categorize different species of animals using Convolutional Neural Networks (CNNs).
*   **`FC25_Rating_prediction.ipynb`**: A predictive modeling project estimating EA SPORTS FC 25 player ratings based on underlying performance metrics and attributes.
*   **`Grade_regression_classification.ipynb`**: A dual-approach deep learning project applying both regression and classification techniques to predict student grades.
*   **`Image_Captioning_Assignment.ipynb`**: Implementation of an image captioning model combining computer vision (to extract image features) and sequence modeling (to generate descriptive text).
*   **`Image_Colorization_Assignment.ipynb`**: A deep learning model built to automatically add realistic color to grayscale images.
*   **`MLP_vs_CNN_experiment.ipynb`**: An empirical study comparing the performance, training time, and accuracy of Multi-Layer Perceptrons (MLPs) against Convolutional Neural Networks (CNNs) on image data.
*   **`Object_Detection_Assignment.ipynb`**: Implementation of object detection algorithms to identify and draw bounding boxes around objects within images.
